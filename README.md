# Trabajo Práctico N°5: JavaScript 
### Programación II 

##  Descripción
Este proyecto integra diversas funcionalidades desarrolladas con **JavaScript**, manipulando el DOM de forma dinámica y consumiendo datos de una API externa. Se aplicaron conceptos de programación asíncrona, manejo de eventos y diseño responsivo.

##  Funcionalidades Principales

### 1. Ejercicios de Lógica (Consola)
Resolución de problemas utilizando métodos avanzados de arrays:
- **Map:** Transformación de datos de alumnos.
- **Filter:** Filtrado por condiciones académicas.
- **Reduce:** Cálculo de promedios.
- **Find:** Búsqueda específica de registros.

### 2. Catálogo de Productos
Sistema de filtrado en tiempo real que permite combinar:
- Búsqueda por nombre.
- Filtrado por categoría.
- Rango de precios y disponibilidad de stock.

### 3. To-Do App
Una lista de tareas interactiva que incluye:
- Agregar nuevas tareas (validación de campos vacíos).
- Marcar tareas como completadas (Toggle).
- Eliminar tareas individualmente.
- Contador dinámico de tareas pendientes.

### 4. PokéAPI Explorer
Consumo de la [PokéAPI](https://pokeapi.co/) con las siguientes características:
- Carga asíncrona con estados de "Cargando..." y manejo de errores.
- Renderizado de tarjetas dinámicas con Grid CSS.
- Buscador dinámico con requisito de mínimo 3 caracteres para filtrar.

##  Tecnologías Utilizadas
- **HTML5** y **CSS3** (Uso de Variables, Flexbox y Grid).
- **JavaScript (ES6+):** Async/Await, Arrow Functions, Template Literals.
- **Git & GitHub:** Control de versiones con flujo de ramas.
